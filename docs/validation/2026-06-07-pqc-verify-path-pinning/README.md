# PQC Verify Path and vCPU Pinning Validation

Curated validation addendum for the 2026-06-07 PQC verify-capacity session.

This package summarizes service-path checks and single-VM CPU placement observations for ML-DSA and SLH-DSA verify workloads.

Included files:

- report.md
- methodology.md
- manifest.json
- summaries/noop_path_summary.csv
- summaries/pinning_verify_summary.csv
- summaries/pinning_delta_summary.csv

Large generated signature pools, raw terminal logs, and local run directories are intentionally excluded.
